# 문제 흐름 분석

## 1. 입력 정보
- 어떤 값들이 주어지는가?
  - 첫째 줄에 기타의 개수 N과 곡의 개수 M (N ≤ 10, M ≤ 50)
  - 둘째 줄부터 N개의 줄에 기타의 이름과 기타가 연주할 수 있는 곡의 정보가 1번 곡부터 차례대로 주어짐

## 2. 출력 정보
- 필요한 기타의 개수를 출력

## 3. 핵심 조건 및 규칙
- 기타의 이름은 알파벳 대문자로만 이루어져 있고, 길이는 2 이상, 50 이하
- 두 기타의 이름이 같은 경우는 없음
